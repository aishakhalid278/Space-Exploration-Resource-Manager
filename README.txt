Chosen Problem: Space Exploration Resource Manager
Description: 
This project implements a system to manage life-support and propulsion resources 
for a spacecraft. It uses operator overloading to calculate resource consumption 
and logistics, and function overloading to handle different types of supply 
intake. The logic is encapsulated in a standalone C++ library.
